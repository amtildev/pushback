# Changelog

## 0.1.0

- Initial public-ready release.
- Adds the `pushback` Codex skill.
- Adds install instructions, usage examples, validation script, and GitHub Actions workflow.

## 0.1.1

- Adds a visual README banner and compact project mark.
- Improves README hierarchy with badges, tables, and clearer examples.

## 0.2.0

- Adds a universal Pushback prompt for non-Codex assistants.
- Adds before/after demos built for screenshots and launch posts.
- Adds launch copy, submission targets, and a social card for distribution.
- Updates the README to lead with the broader "AI agrees too much" hook.

## 0.3.0

- Adds direct adapters for Claude Code, Cursor, Gemini CLI, GitHub Copilot, Cline, Roo Code, Kilo Code/KiloClaw, Hermes, OpenCode, OpenClaw-style agents, Windsurf, Aider, and AGENTS.md-compatible agents.
- Adds `scripts/install.py` to install selected adapters into another project.
- Adds `docs/agent-support.md` with per-agent file mapping and references.
