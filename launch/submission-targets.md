# Submission Targets

Pushback is more likely to spread through places where agent users already look for tools.

## Highest Priority

| Channel | Why it matters | Suggested angle |
|---|---|---|
| Awesome Codex Skills | Direct fit for the repo | A tiny skill that makes Codex challenge assumptions before agreeing |
| Awesome Claude Skills | Larger adjacent audience | Universal prompt plus Codex skill, useful for Claude-style project rules |
| Hacker News Show HN | Good fit for small dev tools | Truth-seeking mode for coding agents |
| X dev/AI builders | The pain is memeable | Your AI assistant agrees too much |
| Reddit r/LocalLLaMA or r/ChatGPTCoding | Prompt and agent users | Copy-paste prompt for reducing yes-man behavior |

## Outreach Targets

Look for posts, newsletters, or lists about:

- Codex skills
- Claude skills
- AI agent prompts
- prompt engineering
- coding-agent workflows
- AI safety for day-to-day developer tools
